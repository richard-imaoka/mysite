JavaScript's Object is a data type that have properties

You can create an Object, which has so-called properties. 
var book = {
  title: "JavaScript: The Good Parts",
  author: "Douglas Crockford",
  pages: 172 //DO NOT put, for the last property
};

console.log(book);

Object {title: "JavaScript: The Good Parts", author: "Douglas Crockford", pages: 172}
A property consists of a property name, and property value. In the above example, below are the properties:
property name property value
title "JavaScript: The Good Parts"
author "Douglas Crockford"
pages 172

You can access to property values by object.propertyName, or object["propertyName"]

var book = {
  title: "JavaScript: The Good Parts",
  author: "Douglas Crockford",
  pages: 172 //DO NOT put, for the last property
};

console.log(book.title);
console.log(book["title"]);
